"Once upon a time ..."

届かない想い、封じたこと。じわりと残る、その傷跡。
過ぎ去った女の子の時間を、絵画の少女に
投影しました。

大人になるにつれて
たくさんの衣に身を隠して、自分の居場所を
掃いて捨てて、の繰り返し。

そうして、ぬけがらになった私たち。
何もない？ いいえ、感じる？
ぬくもりは、いつまで宿る？

How to use

ご入場方法
本ページ下部の扉から, Mozilla Hubsが立ち上がります。以下設定後, ご入場ください
  1. Join Roomをクリック
  2. Display Nameとアバターを選択後、Acceptをクリック
  3. Audioのテスト後、Enter Roomでご入場いただけます
※閲覧環境により見え方が異なります。Mobileでは正常にご覧いただけません。

推奨環境：PC
ブラウザ：Google Chrome 最新版 / Firefox 最新版

デスクトップモードでご入場の方の操作方法
キーボード ←↑↓→：移動
Qキー：左へ向く
Eキー：右へ向く
マウスドラッグ：視点変更
